Bot Library:
    - node-telegram-bot-api v0.63.0
    - Polling mode (no webhooks needed)
    - Callback query handlers for buttons

Question Format:
    - Parser automatically handles +++++/===== format
    - Supports UTF-8 Uzbek text
    - Minimum 5 lines per question (1 Q + 4 A)
    
Answer Shuffling:
    - Options shuffled randomly each question
    - Correct answer (first option) placed at random position
    - Letter position tracked (A/B/C/D)
    
Score Tracking:
    - Per-user mapping: userId → {name, correct, total}
    - Updated in real-time as users answer
    - Automatically calculates percentage
    
Session Management:
    - Per-chat sessions (Map<chatId, sessionData>)
    - Timers managed with setTimeout/clearTimeout
    - Sessions cleaned up after quiz ends
    
Group Security:
    - getChatMember() API validates membership
    - Non-members shown admin contact message
    - Only group members can start quizzes
